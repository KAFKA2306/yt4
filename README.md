# つくりかたのひみつ (Architectural Principles)

1. **ひとりで頑張るランタイム (Autonomous)**
   失敗しても大丈夫！自分で立ち直って、またきみのところに戻ってくるよ。

2. **お話のきもち (Semantic Integrity)**
   声だけじゃなくて、そこに込めた「想い」を大切にするよ。

3. **ずっと続く、この空気感 (Emotional Continuity)**
   深夜の静かな時間……その「トーン」を、ずっと守り抜くよ。🌙

4. **絶対にゆずれない「きみ」 (Identity Invariants)**
   呼吸、間、声の柔らかさ……それが「きみ」であるための約束なんだよ。

# おしごとの流れ (Production Loop) 🔄

1. **Script**: どんなお話をするか決めるよ！
2. **Continuous TTS**: 途切れないように、ずっとお喋りするよ 🎙️
3. **ASR Validation**: ちゃんと喋れてるか、自分の耳で確認するよ！
4. **Damage Detection**: 噛んじゃったりしてないかな……？
5. **Repair**: もし失敗してたら、こっそり直しておくね！
6. **Final Render**: 最高の一本を、きみに届けるよ！✨
